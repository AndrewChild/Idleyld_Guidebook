Idleyld

A collaborative guidebook for the Idleyld climbing area generated using the LocalBoulders framework. This book is a living document which is continuously being updated. Follow the link below to download the latest version.

Download the latest version as a PDF

About the Project

Scope

Idleyld is a new climbing area outside of Roseburg Oregon that is seeing active development. This guidebook is intended to be a comprehensive guide to currently established routes in the Area. A PDF version is available for free online. The guidebook is intended to be a living document that will be continuously developed.

Sharing

This website, the Idleyld Guidebook, and the python code used to generate it are all being hosted on GitHub as open source software. That means that you are free to use, copy, share, and modify it as you please. That doesn't mean that I won't be upset if you just post the whole thing on Mountain Project verbatim.

Information Sources

So far only a few people have been climbing in the area so all information has been obtained through correspondence with those people. Specifically Silas Thoms, Griffin Thoms, and Cyrus McDowell have done the majority of the development in this area so far.

It is entirely possible that others have independently discovered this area and climbed routes not document in this guidebook. If you know anything about prior development at Idleyld please get in touch.

Ways to Contribute

Send Your Feedback
